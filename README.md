# Cybersecurity Learning Journey

A collection of rewritten notes, glossaries, summaries, and study materials from a beginner who is building a foundation in IT and cybersecurity from scratch.

## Project Goal

This repository was created with a very simple purpose:

- to study in an organized way;
- to turn what I learn into clear and reusable material;
- to build a real technical foundation over time;
- to create a public record of my learning journey.

## Starting Point

I am following a cybersecurity learning path while still building my foundational knowledge.

For this reason I will publish materials consistent with the approach of a serious beginner:

- basic concepts;
- technical glossaries;
- simplified summaries;
- notes on systems, networks, Linux, and security;
- small mistakes, corrections, and lessons learned.

## What You Will Find Here

This repository will gradually include:

- technical English glossary
- notes on IT fundamentals
- notes on networks and protocols
- basic operating system concepts
- Linux notes
- cybersecurity summaries
- lab notes
- weekly progress reviews

## Current Status

This repository is still in its early stage.
At the moment, it contains the initial structure of the project, a growing glossary, and the first preparation notes.
More content will be added progressively as the learning journey continues.

## Method

The material published here follows a simple approach:

- understand the topic;
- rewrite it in simple words;
- remove unnecessary complexity;
- keep only what is useful for review and long term learning.

The goal is to build real understanding over time.

## Why This Project Exists

Because learning better also means:

- writing;
- reorganizing;
- simplifying;
- reviewing mistakes;
- building consistency.

This repository exists to support that process.
