# Security Basics

This folder contains introductory notes on cybersecurity concepts and defensive security principles.

It is intended as a beginner friendly space for understanding the logic behind security, rather than collecting advanced or overly technical material too early.

## What this folder should contain

- basic cybersecurity concepts
- threats, vulnerabilities, and risk
- authentication and access control
- least privilege and basic hardening
- attack surface and exposure
- common security mistakes
- simple defensive security principles
- introductory security terminology

## Goal

The goal of this section is to build a clear understanding of the basic concepts that support more advanced cybersecurity learning later on.
