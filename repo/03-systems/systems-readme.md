# Systems

This folder contains notes on system related concepts, with a focus on how operating systems and system resources are organized and managed.

It includes beginner level material useful for understanding both administration and security basics.

## What this folder should contain

- system structure and components
- users and groups
- permissions and access control
- processes and services
- logs and event monitoring
- system configuration basics
- introductory notes on Windows and Linux system behavior

## Goal

The goal of this section is to understand how systems work internally and how they are managed, monitored, and protected in real environments.
