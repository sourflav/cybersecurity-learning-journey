# Fundamentals

This folder contains notes on the core IT concepts that support further learning in cybersecurity.

These notes are meant to help build a solid foundation before moving into more specific security topics.

## What this folder should contain

- basic computer concepts
- hardware and software basics
- operating system fundamentals
- files, folders, and file systems
- users, permissions, and system structure
- simple explanations of how computers work

## Goal

The goal of this section is to strengthen the basic knowledge required to study networking, Linux, systems administration, and cybersecurity more effectively.
