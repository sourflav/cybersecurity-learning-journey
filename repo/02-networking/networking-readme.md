# Networking

This folder contains notes on networking fundamentals and related concepts.

Networking is one of the most important areas to understand before approaching cybersecurity in a meaningful way.

## What this folder should contain

- basic network concepts
- IP addresses and subnet basics
- routers, switches, and gateways
- DNS and DHCP
- ports and protocols
- local networks and internet basics
- simple explanations of how devices communicate

## Goal

The goal of this section is to build a clear understanding of how networks work, so that security concepts can be studied with more context and less confusion.
