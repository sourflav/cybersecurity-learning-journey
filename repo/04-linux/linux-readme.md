# Linux

This folder contains notes, commands, and practice material related to Linux.

Linux is a fundamental area for anyone studying IT, system administration, or cybersecurity.

## What this folder should contain

- basic Linux concepts
- terminal and shell notes
- essential commands
- file and directory navigation
- permissions and ownership
- package management basics
- small command line exercises
- personal notes from practice sessions

## Goal

The goal of this section is to become more comfortable with Linux environments, command line usage, and the basic skills needed for further technical study.
